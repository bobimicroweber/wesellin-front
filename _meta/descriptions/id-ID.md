## Buat situs Anda lebih cepat dan ramah perangkat seluler dengan Ekstensi Plesk Google PageSpeed Insights

Kunci untuk mendapatkan visibilitas yang baik dalam mesin pencari adalah membuat situs web yang cepat, bermanfaat untuk pengunjung Anda, dan dioptimalkan untuk semua trafik, khususnya trafik perangkat seluler, alat Google PageSpeed Insights menganalisis konten situs web dan kinerjanya untuk mengusulkan peningkatan spesifik.

** Cara kerja ekstensi Google PageSpeed Insights:**

- Menganalisis performa situs web yang di-host di server Plesk Anda.
- Memberi skor pada setiap situs web desktop dan seluler yang dianalisis sesuai dengan performanya.
- Membuat laporan berdasarkan hasil analisis.
- Menampilkan saran untuk mengoptimalkan kinerja situs web Anda.
- Memberikan tautan dalam UI ekstensi ke alat bantu yang disarankan untuk meningkatkan performa situs web (misalnya, modul Apache *mod_pagespeed*).
- Menyediakan berkas yang sudah dikompres untuk mengurangi ukuran berkas statis (diperlukan kunci API gratis).
- Memasang modul Apache *mod_pagespeed* dan memberi tahu Anda cara mengonfigurasinya untuk kebutuhan Anda.
